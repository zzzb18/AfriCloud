# ☁️ AI Cloud Storage 使用说明

## 🚀 快速启动

### 方法1: 使用批处理文件（推荐）
双击 `start_app.bat` 文件即可启动应用

### 方法2: 使用命令行
```bash
python -m streamlit run app.py
```

### 方法3: 使用Python脚本
```bash
python run_app.py
```

## 🌐 访问应用

启动成功后，在浏览器中访问：
**http://localhost:8501**

## 📋 功能说明

### 1. 文件上传
- **普通上传**: 适合小文件和稳定网络
- **断点续传**: 适合大文件和不稳定网络
- **支持格式**: Excel、PDF、图片、CSV、文本等

### 2. 文件管理
- **预览文件**: 点击👁️按钮预览文件内容
- **缓存文件**: 点击💾按钮将文件保存到本地
- **下载文件**: 点击📥按钮下载文件

### 3. 文件夹管理
- 在侧边栏输入文件夹名称
- 点击"创建"按钮创建新文件夹

### 4. 搜索功能
- 在侧边栏输入关键词搜索文件
- 可以按文件类型筛选结果

### 5. 上传进度
- 实时显示上传进度
- 支持暂停和恢复上传
- 支持取消上传

## 🔧 故障排除

### 问题1: 无法启动应用
**解决方案**:
```bash
pip install -r requirements.txt
python -m streamlit run app.py
```

### 问题2: PDF预览不工作
**解决方案**:
```bash
pip install PyMuPDF
```

### 问题3: 端口被占用
**解决方案**:
```bash
python -m streamlit run app.py --server.port=8502
```

### 问题4: 文件上传失败
**检查项目**:
- 确保有足够的磁盘空间
- 检查文件格式是否支持
- 尝试使用断点续传模式

## 📁 文件存储位置

- **云端存储**: `cloud_storage/` 目录
- **本地缓存**: `local_cache/` 目录
- **数据库**: `cloud_storage/storage.db`

## 🎯 使用技巧

1. **大文件上传**: 使用断点续传模式
2. **离线访问**: 提前缓存重要文件
3. **文件整理**: 创建文件夹分类管理
4. **快速查找**: 使用搜索功能定位文件

## 📞 技术支持

如果遇到问题，请检查：
1. Python版本 >= 3.8
2. 已安装所有依赖包
3. 端口8501未被占用
4. 有足够的磁盘空间
